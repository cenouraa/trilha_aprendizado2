Tag < table >

A tag < table > define uma tabela HTML. Uma tabela HTML consiste em um elemento < table >, e um ou mais elementos < tr >, < th > e < td >.

-> O elemento < tr > define uma linha de tabela.
-> O elemento < th > define um cabeçalho de tabela.
-> O elemento < td > define uma célula de tabela.

Uma tabela HTML também pode incluir elementos < caption >, < colgroup >, < thead >, < tfoot > e < tbody >. Ex:

Mês Economias
Janeiro $100

Tag < del >

A tag < del > define texto que foi excluído de um documento. Os navegadores geralmente atacam uma linha através de texto suprimido.

Minha cor favorita é azul!

Tag < ins >

A tag < ins > define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido.

Minha cor favorita é vermelho!

Tag < sub >

A tag < sub > define texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. Texto subscrito pode ser usado para fórmulas químicas, como H2O.

Ex: Esse texto contém texto subscrito.

Tag < sup >

A tag < sup > define texto de sobrescrito. O texto de sobrescrito aparece meio caractere acima da linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado para notas de rodapé, como WWW[1].

Ex: Esse texto contém texto sobrescrito.

Tag < blockquote >

A tag < blockquote > especifica uma seção que é citada de outra fonte.

Ex: Aqui está uma citação do website da WWF:

Por 50 anos, a WWF tem protegido o futuro da natureza. Como a principal organização de conservação do mundo, WWF trabalha em 100 países e é apoiada por 1.2 milhões de membros nos Estados Unidos e perto de 5 milhões no mundo todo.

Tag < q >

A tag < q > define uma curta citação.

Ex: O objetivo da WWF's é: construir um futuro onde pessoas vivam em harmonia com a natureza. Nós esperamos que eles consigam.

Tag < abbr >

A tag < abbr > define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Ex: O IFPR - Campus Irati foi fundado em 23/04/2010.

Tag < address >

A tag < address > define as informações de contato para o autor/proprietário de um documento ou um artigo. As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone, alça de mídia social, etc. O texto, geralmente, é renderizado em itálico, e os navegadores sempre adicionarão uma quebra de linha antes e depois do elemento. Ex:

Escrito por João Developer.
Nos visite em:
Exemplo.com
Caixa 564, Disneylândia
BR

Tag < cite >

A tag < cite > define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).
Nota: O nome de uma pessoa não é o título de uma obra.
O texto no elemento geralmente é renderizado em itálico.

Ex: O Grito por Edward Munch. Pintado por 1893.

Tag < bdo >

BDO é uma sigla para Bi-Directional Override e a tag < bdo > é usada para substituir a direção de texto atual.

Ex: Esse texto irá da direita para a esquerda.

Tag < code >

A tag < code > é usada para definir um treço de código de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

A tag button em HTML define um botão clicável.

Tag < pre >

A tag < pre > define o texto pré-formatado. O elemento é exibido em uma fonte de largura fixa e o texto preserva espaços e quebras de linha. O texto será exibido exatamente como escrito no código-fonte HTML.

Ex:

        Texto em um pré elemento
        é mostrado num tamanho fixo
        de fonte, e mantem
        ambos      os espaços           
        e quebras de linha.

Tag < kbd >

A tag < kbd > é usada para definir um imput do teclado. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Ex: Pressione Ctrl + C para copiar texto (Windows).
