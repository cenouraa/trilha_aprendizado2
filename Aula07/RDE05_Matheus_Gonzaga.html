Hyperlinks

Os hyperlinks são uma referência dentro de um documento em hipertexo para outras partes do mesmo documento, outros documentos, arquivos ou endereço eletrônico. Os links podem ser remotos, internos, locais, com endereço de correio ou arquivos.

Marcação de Hyperlinks

A marcação de hiperlinks acontece utilizando a tag < a > e a propriedade href que faz a referência para ao conteúdo desejado.

- Sintaxe: <a href="endereço"> Conteúdo Referenciado Clicável < /a>

Hyperlinks externos ou remotos

São links que referenciam sites ou páginas da web. Na propriedade href utilizamos a URL (Uniform Resource Locator) para a referência que desejamos.

- Sintaxe: <a href="URL"> Conteúdo Referenciado < /a>

Hiperlinks internos

São links quem apontam a um lugar diferente dentro da mesma página e possibilita acessar diferentes parágrafos ou seções quando o documneto é extenso.

- Sintaxe: <a href="#referencia"> Conteúdo Referenciado < /a> - <a name="referencia"> < /a>

- Sintaxe: <a href="#referencia"> Conteúdo Referenciado < /a> - <tag id="referencia">

Hiperlinks locais

São links que apontam para outras páginas dentro de um mesmo site.

- Sintaxe: <a href="arquivo.html"> Conteúdo Referenciado < /a>

Hiperlinks em imagens

É possível realizar hiperlinks de imagens envolvendo a tag de imagem com a tag de hiperlink.

- Sintaxe: <a href="referencia"> <img src="imagem.jpg"> < /a>

Hiperlinks para endereço eletrônico

São links que apontam para um endereço eletrônico

- Sintaxe: <a href="mailto:email@email.com"> email@email.com < /a>
